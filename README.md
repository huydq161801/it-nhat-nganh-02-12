# Instagram

 được viết bằng ruby on rails

**Các chức năng của website**:
  - Đăng nhập/Đăng ký
  - Đăng bài (có thể đăng nhiều ảnh)
  - Bookmark bài đăng
  
**Hướng dẫn sử dụng**:
1. Install Ruby, Rails.
2. Clone repo từ github về, mở Terminal và thực hiện các lệnh sau:
3. `bundle install`
4. `rake db:migrate`
5. `rails s`
